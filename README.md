## üìå **Project Description**

This project presents a **fully interactive HR Analytics Dashboard** built using **Microsoft Excel**, 
designed to help organizations understand **employee attrition patterns, workforce distribution, and key HR insights**.
Using Pivot Tables, Pivot Charts, Slicers, and KPI cards, the dashboard converts raw HR data of **1,470 employees**
into a clear visual story that helps HR teams make **data-driven retention and workforce planning decisions**.

The dashboard answers critical business questions such as:

* How many employees left the company and why?
* Which age groups exhibit the highest attrition?
* Which departments and job roles are most affected?
* How do education, gender, and marital status influence attrition?
* What is the overall employee satisfaction level?
